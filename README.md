# vehicles-dashboard
Esta aplicación web permite realizar un análisis exploratorio de datos
sobre anuncios de autos en Estados Unidos.

## Funcionalidad
- Carga un dataset de vehículos
- Muestra histogramas interactivos
- Muestra gráficos de dispersión
- Construida con Streamlit y Plotly

## Tecnologías
- Python
- Pandas
- Plotly
- Streamlit
